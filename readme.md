# Tuto jeu de survie 3D

### Commandes

- espace : sauter
- ZQSD : se déplacer
- shift + ZQSD : sprinter
- clic droit : zoom/inspection
- F : fly mode

### Logiciels utilisés

- Unity 2021.3.31f1
- Microsoft Visual Studio Community 2022 (version 17.7.5)
